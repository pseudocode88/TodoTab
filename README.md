# Todo Tab
A simple todo list for those who spend most of the day in front of the browser. No login, no big background image, just a simple Todo list. Featured on Product Hunt and LifeHacker Japan.

[Download from Chrome Store](https://chromewebstore.google.com/detail/todo-tab/ljkjodkdilmmlaiphehiceeblnnndhnd?hl=en-GB&authuser=2)

**Version 2.0.1 - Since 2017**

## Verb Highlighting
> “Identify your tasks faster through activity based color coding”

Todo Tab parses your task description and looks for activities (verb) like call, meeting, reply, etc. and show it in different colours. This feature improves the readability of the list and helps you to recognise as well as recall the context faster.

## Setting up Development

```
$ git clone git@bitbucket.org:27ae60/todo-tab.git
$ npm install
$ npm run watch
```

For compiling the JSX files with out watch: `$ node compile.js`
