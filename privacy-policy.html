Todo Tab

A privacy first to-do app - version 2.0.1

*   *   *
“Privacy Policy”
  1. TodoTab doesn't store and track any of your personal information.
  2. TodoTab is a complete permission less extension.
  3. TodoTab use mixpanel for anaylitics, but no user information or identity is attached.
  4. TodoTab use browser's local storage to persist your data
  5. For analytics, we use hard coded labels only - Add task, Edit task, Finish task, Delete task, Add Activity, Delete Activity, Update to v2.0.1 and Install v2.0.0
